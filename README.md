# lemin-java
Java version of 42 C project lem-in

## How to use it
Compile with\
\> javac Lemin.java\
Run with\
\> java Lemin < maps/[map_file]

## Options
| Option | Effect |
|--------|--------|
|-h --help| Display help|
|-p --print| Echo the input to the standard input|
|-c --color| Colorize ants|
|--ants X| Manually assigns number of ants to X, if X is a valid integer|
